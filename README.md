# Triple M Business Solutions

Marketing site for Triple M Business Solutions — bookkeeping, QBO organization, and back-office infrastructure for small construction and trade companies.

Built with Next.js (App Router) and Tailwind CSS.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Pages

- `/` — Home
- `/services` — Services by category
- `/pricing` — Engagement & pricing model
- `/about` — Founder story & credentials
- `/contact` — Consultation request form

The contact form currently only shows a client-side confirmation state — it isn't wired to a backend or email delivery service yet.
